setCellBorderColor(columnIndex, rowIndex, color:uint,thickness)
==>columnIndex is column index
==>rowIndex is row index
==>color is color code of border
==>thickness is thickness of border

Ex: gridOne.setCellBorderColor(1,0,0x00DE00,1);
removeCellBorderAll()
==> this method will remove all border was setted from dataGrid.

Ex: gridOne.removeCellBorderAll();
removeCellBorder(columnIndex,rowIndex)
==>columnIndex is column index which has setted border.
==>rowIndex is row index which has setted border.

Ex: gridOne.removeCellBorder(0,2);
